# AI Quiz Assistant 

AI Quiz Assistant is a Chrome Extension that helps users interact with online quiz forms more efficiently.

The extension can:
- Extract quiz questions from Google Forms
- Read MCQ options
- Prepare the foundation for AI-based answer suggestions

---

## Features

- 📄 Detects Google Form quiz questions
- ✅ Extracts multiple choice options
- ⚡ Built using Chrome Extension Manifest V3
- 🧠 Future scope for AI-generated answer suggestions

---

## Tech Stack

- JavaScript
- HTML
- CSS
- Chrome Extensions API
